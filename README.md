# 家計簿アプリ

## 概要

支出・収入をカテゴリごとに分けて記録・管理できる家計簿アプリです。  
投稿の作成・編集・削除や月別集計、グラフ表示、テーマカラー変更などの設定を行うことができます。

## 使用技術

- HTML / CSS
- JavaScript
- ローカルストレージ（データ保存用）

## 画面構成・機能紹介

### ホーム画面

- 一覧で支出・収入を確認
- カテゴリごとに絞り込み可能
- 月別収支のサマリー表示

### カレンダー画面

- 月ごとの収支をカレンダー形式で表示
- 日付をクリックするとその日の投稿一覧を確認可能
- 収入・支出の管理が直感的にできる

### グラフ画面

- 月ごとの収支を集計して表示
- カテゴリ別のグラフで支出・収入を可視化
- 期間を選んで集計可能

## 工夫した点

- ローカルストレージを用いてデータをブラウザ内に保存し、オフラインでも使用可能
- JavaScript で状態管理を自作し、入力データや設定の同期を効率化
- シンプルかつ見やすい UI を設計し、ユーザーが直感的に操作できるよう工夫
